
# Emoji Tournament

Ett litet React-projekt som låter dig köra en turnering mellan emojis.  
Användaren kan välja vinnare i varje match tills en slutlig vinnare utses.  
Slutvinnaren sparas även i `localStorage`.

## Kom igång

### Förutsättningar
- Node.js (>=18)
- npm eller yarn

## Installation

Kör `npm install` för att installera alla paket.

## Utveckling

Kör `npm run dev` för att starta utvecklingsservern.

## Testning

Kör `npm test` för att köra testerna. Kan köras utan att utvecklingsservern är igång.

## Coverage

Kör `npm run coverage` för att se hur stor del av kodbasen som testerna täcker.
